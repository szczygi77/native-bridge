# Native Bridge — Landing Page

Prototyp strony głównej szkoły językowej Native Bridge (HTML + CSS + JS, jeden plik).

## Podgląd

Otwórz `index.html` bezpośrednio w przeglądarce (plik musi leżeć w tym samym folderze co `logo.png` i zdjęcia `zdj-*.jpg/png`).

```bash
open index.html
```

## Zawartość

- `index.html` — kompletna strona (hero, Metoda 4D, kursy, lektorzy, cennik, quiz, kontakt)
- `logo.png`, `zdj-2.jpg`, `zdj-3.jpg`, `zdj-4.png` — zasoby graficzne
